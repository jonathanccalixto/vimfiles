# vimfiles

Cross-platform vimfiles (Mac OS X and Linux)

## Usage

``` bash
  path=$(pwd)/Configurations/vimfiles

  git clone git@github.com:jonathanccalixto/vimfiles.git $path

  ln -s $path ~/.vim
  ln -s $path/vimrc ~/.vimrc
```
